This zip file contains Comma Separated Value (CSV) files.  The CSV files may be opened in a 
text editor such as Notepad.  The following instructions explain how to import CSV files into Microsoft Excel.

1. Open a new blank spreadsheet in Excel.

2. Select "data", "get external data", "from text", and select the CSV file.

3. Select "delimited" from the first screen of the wizard.

4. Select "comma" from the second screen.

5. On the third screen, change all fields to "text". 

6. Proceed with the import.
